---
title: useLeaveOrganization
description: Leave an organization.
---

Organization mutations require `organizationClient()` on the Better Auth client.

## Usage

```tsx
import { useAuth } from "@better-auth-ui/react"
import { useLeaveOrganization } from "@better-auth-ui/react/plugins/organization"

const { authClient } = useAuth()
const mutation = useLeaveOrganization(authClient)

mutation.mutate(/* params */)
```

## Options factory

```tsx
import { leaveOrganizationOptions } from "@better-auth-ui/react/plugins/organization"
import { useMutation } from "@tanstack/react-query"

const mutation = useMutation(leaveOrganizationOptions(authClient))
```
