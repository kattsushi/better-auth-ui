Solid mutations expose `@tanstack/solid-query` mutation options for Better Auth write endpoints. They preserve shared mutation keys and pass `throw: true` so failures surface through standard Solid Query error handling.

## Quick path

1. Import the specific `*Options` factory from `@better-auth-ui/solid`.
2. Pass the matching Better Auth client method through your typed `authClient`.
3. Use the returned options with `createMutation`.

```tsx
import { signInEmailOptions } from "@better-auth-ui/solid"
import { createMutation } from "@tanstack/solid-query"

const signIn = createMutation(() => signInEmailOptions(authClient))
```

## Auth mutations

| Concern | Helpers |
| --- | --- |
| Email auth | `signInEmailOptions`, `signUpEmailOptions`, `signOutOptions`. |
| Social auth | `signInSocialOptions`, `linkSocialOptions`, `unlinkAccountOptions`. |
| Password flow | `requestPasswordResetOptions`, `resetPasswordOptions`, `changePasswordOptions`. |
| Email verification | `sendVerificationEmailOptions`, `changeEmailOptions`. |

## Settings mutations

Use `updateUserOptions`, `deleteUserOptions`, and `revokeSessionOptions` for account settings flows. These are package-level API wrappers; copied settings screens belong to Zaidan.

## Plugin-backed mutations

| Plugin capability | Helpers |
| --- | --- |
| Username | `signInUsernameOptions`, `isUsernameAvailableOptions`. |
| Magic link | `signInMagicLinkOptions`. |
| Passkey | `addPasskeyOptions`, `deletePasskeyOptions`, `signInPasskeyOptions`. |
| Multi-session | `setActiveSessionOptions`, `revokeMultiSessionOptions`. |
| API key | `createApiKeyOptions`, `deleteApiKeyOptions`. |

## Generic primitive

`authMutationOptions` and `createAuthMutationOptions` are lower-level adapters for Better Auth client methods. Prefer the named factories in app code; use the primitives when adding a new package mutation family.

## UI boundary

The Solid package does not render toast UI. Attach success/error handling in your app, or use installed Zaidan components that already integrate with the app's toast surface.
