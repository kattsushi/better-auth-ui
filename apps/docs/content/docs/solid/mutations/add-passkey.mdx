---
title: addPasskeyOptions
description: Register a new passkey for the current user.
---

Requires the Better Auth passkey plugin.

## Usage

```tsx
import { addPasskeyOptions } from "@better-auth-ui/core/plugins/passkey"
import { createMutation } from "@tanstack/solid-query"

const addPasskey = createMutation(() => addPasskeyOptions(authClient))

addPasskey.mutate({ name: "Work laptop" })
```
