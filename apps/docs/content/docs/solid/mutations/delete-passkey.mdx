---
title: deletePasskeyOptions
description: Delete a passkey from the current user's account.
---

Requires the Better Auth passkey plugin.

## Usage

```tsx
import { deletePasskeyOptions } from "@better-auth-ui/solid/plugins/passkey"
import { createMutation } from "@tanstack/solid-query"

const deletePasskey = createMutation(() => deletePasskeyOptions(authClient))

deletePasskey.mutate({ id: "passkey-id" })
```
