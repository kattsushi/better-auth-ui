---
title: useSetActiveSession
description: Switch the active session in a multi-session setup.
---

Requires the Better Auth multi-session plugin.

## Usage

```tsx
import { useSetActiveSession } from "@better-auth-ui/solid/plugins/multi-session"

const setActiveSession = useSetActiveSession(authClient)

setActiveSession.mutate({ sessionToken: "session-token" })
```

## Options factory

```tsx
import { setActiveSessionOptions } from "@better-auth-ui/core/plugins/multi-session"
import { createMutation } from "@tanstack/solid-query"

const setActiveSession = createMutation(() =>
  setActiveSessionOptions(authClient, userId)
)
```

## Cache

The core options metadata refetches the session and device-session queries after switching.
