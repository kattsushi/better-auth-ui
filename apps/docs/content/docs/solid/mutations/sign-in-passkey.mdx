title: signInPasskeyOptions
description: Passkey sign-in mutation options.
---

Requires the Better Auth passkey plugin. When reading the client from auth context, call `useAuth<PasskeyAuthClient>()` so TypeScript sees the plugin methods.

## Options factory

```tsx
import { signInPasskeyOptions } from "@better-auth-ui/solid/plugins/passkey"
import { createMutation } from "@tanstack/solid-query"

const signInWithPasskey = createMutation(() => signInPasskeyOptions(authClient))

signInWithPasskey.mutate()
```
