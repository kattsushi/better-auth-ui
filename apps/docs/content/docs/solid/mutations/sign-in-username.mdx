description: Username/password sign-in mutation options.
---

Requires the Better Auth username plugin.

## Usage

```tsx
import { signInUsernameOptions } from "@better-auth-ui/solid"
import { createMutation } from "@tanstack/solid-query"

const signIn = createMutation(() => signInUsernameOptions(authClient))

signIn.mutate({ username: "alice", password: "hunter2" })
```
