---
title: signInUsernameOptions
description: Username/password sign-in mutation options.
---

Requires the Better Auth username plugin. When reading the client from auth context, call `useAuth<UsernameAuthClient>()` so TypeScript sees the plugin methods.

## Options factory

```tsx
import { signInUsernameOptions } from "@better-auth-ui/solid/plugins/username"
import { createMutation } from "@tanstack/solid-query"

const signIn = createMutation(() => signInUsernameOptions(authClient))

signIn.mutate({ username: "alice", password: passwordInput })
```
