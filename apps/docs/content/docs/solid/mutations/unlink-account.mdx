---
title: unlinkAccountOptions
description: Unlink a social account from the current user.
---

## Usage

```tsx
import { unlinkAccountOptions } from "@better-auth-ui/solid"
import { createMutation } from "@tanstack/solid-query"

const unlinkAccount = createMutation(() => unlinkAccountOptions(authClient))

unlinkAccount.mutate({ providerId: "github", accountId: "github-account-id" })
```

## Cache

Refetch linked accounts after unlinking.
