title: updateUserMutation
description: Update the current user's profile fields.
---

## Usage

```tsx
import { updateUserMutation } from "@better-auth-ui/solid"

const updateUser = updateUserMutation(authClient)

updateUser.mutate({ name: "Alice" })
```

## Cache

The shared mutation invalidator awaits session invalidation after a successful update so cached user data refreshes.
