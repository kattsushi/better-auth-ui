Registry ownership moved to [Zaidan registry](/docs/zaidan/registry). Keep this page as a stable handoff for older `/docs/solid/registry` links while the Solid section refocuses on package and runtime APIs.

The Solid/Zaidan registry is generated from `examples/start-solid-zaidan-example/**` and served as static assets from `apps/docs/public/r/solid`. The docs app only hosts those files; it does not import or execute Solid components.

## Where to go next

- Use [Zaidan registry](/docs/zaidan/registry) for registry endpoints, payload inventory, generation rules, and install commands.
- Use [Zaidan components](/docs/zaidan/components) for copied component payloads.
- Use [Zaidan plugins](/docs/zaidan/plugins) for copied plugin UI payloads.
- Use [Solid](/docs/solid) for package/runtime APIs.

## Source-of-truth rule

When component source changes, update `examples/start-solid-zaidan-example/src/components/**` first and regenerate the Solid registry. Do not hand-edit generated JSON under `apps/docs/public/r/solid`.
