---
title: <AdditionalField />
description: Install shared Solid additional-field rendering.
---

```bash
zaidan add https://better-auth-ui.com/r/solid/additional-field.json
```

Installs the shared field renderer used by sign-up and profile surfaces.

## What it copies

- `src/components/auth/additional-field.tsx`
- Local button, card, input, and label primitives.

After install, adapt validation messages and field layouts to match your Better Auth additional fields.
