---

## Usage

Render `<ForgotPassword />` on the password reset request route.

```tsx
import { ForgotPassword } from "@/components/auth/forgot-password"

export function ForgotPasswordRoute() {
  return <ForgotPassword redirectTo="/auth/reset-password" />
}
```

The component submits through `requestPasswordResetOptions`, shows inline form feedback, and links back to the configured sign-in path.

## Installation

```bash
zaidan add https://better-auth-ui.com/r/solid/forgot-password.json
```

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `redirectTo` | `string` | Optional URL Better Auth should include in the reset email. |

## Installed files

Installs the email request step for password reset flows.

- `src/components/auth/forgot-password.tsx`
- Local button, card, input, and label primitives.

After install, connect email delivery through your Better Auth reset-password setup.
