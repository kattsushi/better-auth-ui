---
title: <ResetPassword />
description: A Solid reset-password form that sets a new password from a reset token.
---

## Usage

Render `<ResetPassword />` on the route that receives reset links. Pass a token explicitly when your router parses it for you.

```tsx
import { ResetPassword } from "@/components/auth/reset-password"

export function ResetPasswordRoute(props: { token?: string }) {
  const token = props.token

  return <ResetPassword token={token} />
}
```

When `token` is not provided, the installed component reads `token` from `window.location.search`, validates matching password fields, and submits through `resetPasswordOptions`.

## Installation

```bash
zaidan add https://better-auth-ui.com/r/solid/reset-password.json
```

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `token` | `string` | Optional reset token. If omitted, the component reads the `token` query parameter in the browser. |

## Installed files

Installs the reset-password form that reads the token from app routing and submits through Solid mutations.

- `src/components/auth/reset-password.tsx`
- Local button, card, input, and label primitives.

After install, verify the route that receives reset links and tune success/error copy.
