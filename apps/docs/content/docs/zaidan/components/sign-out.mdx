---
title: <SignOut />
description: A Solid sign-out component that signs the user out on mount and navigates back to sign-in.
---

## Usage

Render `<SignOut />` on the sign-out route. It calls the installed auth client when mounted.

```tsx
import { SignOut } from "@/components/auth/sign-out"

export function SignOutRoute() {
  return <SignOut label="Signing you out…" />
}
```

After the sign-out request finishes, the component navigates to the configured sign-in path from `<AuthProvider />`.

## Installation

```bash
zaidan add https://better-auth-ui.com/r/solid/sign-out.json
```

## Props

| Prop | Type | Description |
| --- | --- | --- |
| `label` | `string` | Optional status text shown while the sign-out request is running. |

## Installed files

Installs a sign-out action that calls the Solid auth client from the installed app-side provider context.


- `src/components/auth/sign-out.tsx`

After install, decide where users should land after sign-out and adapt navigation if needed.
